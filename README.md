# CRUD System

Sistema desktop desenvolvido em C# .NET com Windows Forms.

## Funcionalidades
✔ Cadastro de usuários  
✔ Edição de registros  
✔ Exclusão  
✔ Busca dinâmica  
✔ Máscaras para CPF e telefone  

## Tecnologias
- C#
- .NET
- Windows Forms
